# esp32-4wd-bluetooth-car

## Overview
This project involves designing and building a high-speed 4WD robotic car using ESP32 with Bluetooth control.

## Features
- Wireless control using ESP32
- High-speed motor operation
- Efficient power management

## Components Used
- ESP32
- Motor Driver (L298N)
- DC Motors
- Battery Pack

## Working
The ESP32 receives Bluetooth commands from a mobile device and controls motor movement accordingly.

## Applications
- Robotics
- Remote-controlled vehicles

## Future Improvements
- Add obstacle detection
- Improve speed control
